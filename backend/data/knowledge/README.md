## medical_guideline.txt
Title: Medical Guidelines
Category: guideline

発熱の目安:
- 37.5度以上: 経過観察
- 38度以上が2日以上: 受診推奨
- 39度以上: 早めの受診

緊急症状（119番または救急受診）:
- 激しい胸痛
- 意識障害
- 大量出血
- 呼吸困難
- .stroke症状（顔の歪み、腕の脱力、言語障害）
